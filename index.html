Assignment

构造NextDate问题的弱一般的等价类测试用例。

NextDate问题：NexDate()是整形变量month,day和year的函数，输入1812-2012年期间的某一日期的month,day和year的值，输出这一天的下一天的日期的month,day和year值。

  • 等价类划分: 一种典型的黑盒测试方法，是把所有可能的输入数据划分成若干部分，然后从每一部分选取少数有代表性的数据作为测试用例
  • 有效等价类：对于程序规格说明来说是合理的，有意义的输入数据构成的集合。
  • 无效等价类：对于程序规格说明来说是不合理的，无意义的输入数据构成的集合。
  • 弱一般: 针对单缺陷的等价类测试用例。

Solution

1. 划分等价类

设年月日分别为Y，D，M
Y1 = {Y: Y属于闰年}
Y2 = {Y: Y不属于闰年}
M1 = {1, 3, 5, 7, 8, 10, 12}
M2 = {4, 6, 9, 11}
M3 = {2}
D1 = {D: 1 < D <= 28}
D2 = {29}
D3 = {30}
D4 = {31}

Solution

用例 Y M D 期望输出
WN1 1996 2 28 1996/3/1
WN2 1998 2 29 1998/3/1
WN3 1996 4 30 1996/5/1
WN4 1998 5 31 1998/6/1

Assignment

构造下述三角形问题的弱健壮的等价类测试用例。

三角形问题：输入一个不超过100的正整数作为三角形的三条边，判断三角形是等边三角形、等腰不等边三角形、完全不等边三角形还是不能构成三角形。

  • 等价类划分: 一种典型的黑盒测试方法，是把所有可能的输入数据划分成若干部分，然后从每一部分选取少数有代表性的数据作为测试用例
  • 有效等价类：对于程序规格说明来说是合理的，有意义的输入数据构成的集合。
  • 无效等价类：对于程序规格说明来说是不合理的，无意义的输入数据构成的集合。
  • 弱健壮性: 针对单缺陷的等价类测试用例，同时覆盖有效等价类和无效等价类的测试用例。

Solution

1. 划分等价类

设输入的三条分别为A、B、C，其中0 < A、B、C <= 100。
则可以划分为

  • R1 = {<A,B,C>:构成等边三角形}
  • R2 = {<A,B,C>:构成等腰不等边三角形}
  • R3 = {<A,B,C>:构成完全不等边三角形}
  • R4 = {<A,B,C>:不能构成三角形}

2. 测试用例

满足弱等价类测试

用例 A B C 期望输出
WN1 5 5 5 等边三角形
WN2 5 5 6 等腰不等边三角形
WN3 3 4 5 完全不等边三角形
WN4 3 3 6 不能构成三角形

满足健壮性等价类测试
