reduce

接收一个函数作为累加器，数组中的每个值（从左往右）开始缩减，最终计算为一个值。

参数 描述
total 必需。初始值，或者上一次调用的返回值
currentValue 必需。当前元素
currentIndex 可选。当前元素的索引
arr 可选。当前元素所属的数组对象
