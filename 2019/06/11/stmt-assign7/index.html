Assignment

构造下述三角形问题的弱健壮的等价类测试用例。

三角形问题：输入一个不超过100的正整数作为三角形的三条边，判断三角形是等边三角形、等腰不等边三角形、完全不等边三角形还是不能构成三角形。

  • 等价类划分: 一种典型的黑盒测试方法，是把所有可能的输入数据划分成若干部分，然后从每一部分选取少数有代表性的数据作为测试用例
  • 有效等价类：对于程序规格说明来说是合理的，有意义的输入数据构成的集合。
  • 无效等价类：对于程序规格说明来说是不合理的，无意义的输入数据构成的集合。
  • 弱健壮性: 针对单缺陷的等价类测试用例，同时覆盖有效等价类和无效等价类的测试用例。

Solution

1. 划分等价类

设输入的三条分别为A、B、C，其中0 < A、B、C <= 100。
则可以划分为

  • R1 = {<A,B,C>:构成等边三角形}
  • R2 = {<A,B,C>:构成等腰不等边三角形}
  • R3 = {<A,B,C>:构成完全不等边三角形}
  • R4 = {<A,B,C>:不能构成三角形}

2. 测试用例

满足弱等价类测试

用例 A B C 期望输出
WN1 5 5 5 等边三角形
WN2 5 5 6 等腰不等边三角形
WN3 3 4 5 完全不等边三角形
WN4 3 3 6 不能构成三角形

满足健壮性等价类测试

用例 A B C 期望输出
WR1 -1 5 5 A小于有效范围
WR2 5 -1 -1 B小于有效范围
WR3 5 5 -1 C小于有效范围
WR4 101 5 5 A大于有效范围
WR5 5 101 5 B大于有效范围
WR6 5 5 101 C大于有效范围
