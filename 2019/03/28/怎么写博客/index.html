2.Front-matter

Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量。如果用上面的命令新建的话，会帮你生成title和date。

查看官方中文使用文档可以知道，还有以下的用法。

参数 描述 默认值
title 布局
title 标题
date 建立日期 文件建立日期
updated 更新日期 文件更新日期
comments 开启文章的评论功能 true
tags 标签（不适用于分页）
categories 分类（不适用于分页）
permalink 覆盖文章网址

之前用csdn写标题都是有特别的输入框，写标签和分类都是在点击发布的时候再添加的，而这里要在Front-Matter里面写好。这样，hexo就会自动分类好的。

5.本地预览图片

用上面的方法插入图片是好，但在VSCode里面看不到效果啊，只有代码。我用的插件是markdown-preview-enhanced。很强大的插件，看看它的文档就能知道它能做什么了。

这个插件还支持扩展md解析语法，只要我们扩展一下就好了。

查看->命令面板->搜索”markdown preview enhanced: extend parser”->在parse.js中修改onWillParseMarkdown方法。

这样就完美支持了。
