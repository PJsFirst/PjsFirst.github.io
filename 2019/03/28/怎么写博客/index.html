2.Front-matter

Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量。如果用上面的命令新建的话，会帮你生成title和date。

查看官方中文使用文档可以知道，还有以下的用法。

参数 描述 默认值
title 布局
title 标题
date 建立日期 文件建立日期
updated 更新日期 文件更新日期
comments 开启文章的评论功能 true
tags 标签（不适用于分页）
categories 分类（不适用于分页）
permalink 覆盖文章网址

之前用csdn写标题都是有特别的输入框，写标签和分类都是在点击发布的时候再添加的，而这里要在Front-Matter里面写好。这样，hexo就会自动分类好的。

3.图片的插入

在csdn图片插入很简单，只要把本地图片上传到csdn，然后它就会返回图片的url，直接用markdown的语法。

虽说你也可以先上传到某个图床得到url，然后再用相同的方法写，但这也太麻烦了。所以这里先试图用本地的图片的相对地址。

把post_asset_folder设为true
