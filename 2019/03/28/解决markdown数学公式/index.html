发现还是不行，别急，仔细阅读里面的英文注释。要在博客里面加 mathjax: true才会加载mathjax。如果默认每页都加载的话，就把 per_page设为 false。

试试效果

一行公式

$$
\Gamma(z) = \int_0^\infty t^{z-1}e^{-t}dt\,.
$$

行内公式

Gamma公式展示 $\Gamma(n) = (n-1)!\quad\forall n\in\mathbb N$

更复杂公式

$$
P = \frac
{\sum_{i=1}^n (x_i- x)(y_i- y)}
{\displaystyle \left[
\sum_{i=1}^n (x_i-x)^2
\sum_{i=1}^n (y_i-y)^2
\right]^{1/2} }
$$
